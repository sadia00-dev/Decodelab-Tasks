# Responsive Web Layout Project

## Overview

This project is a modern responsive webpage built using HTML5 and CSS3.

The webpage includes:

- Responsive Navigation Bar
- Hero Section with Background Image
- CSS Grid Layout with Custom Card Arrangement
- Footer Section


The project demonstrates the use of modern CSS techniques including Grid Layout, Flexbox and Visual Hierarchy.

---

## Features

✅ Responsive Navigation Bar

✅ Hero Section with CTA Button

✅ CSS Grid-based Content Layout

✅ Mixed Card Sizes using Grid Row Positioning

✅ Responsive Images

✅ Footer Section

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox

# 🚀 Interactive Web Elements Project

A modern, responsive JavaScript project featuring interactive UI components like theme toggle, counter controls, and dynamic message display.

## ✨ Features

- 🌙 Dark/Light Mode Toggle (ON/OFF switch style)
- ➕ Increment / ➖ Decrement Counter
- 🔄 Reset Counter Button
- 💬 Show / Hide Message Feature
- 🎨 Clean Responsive UI
- ⚡ Smooth JavaScript Interactions

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
